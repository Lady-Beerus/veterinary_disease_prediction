# Veterinary Disease Prediction

## Overview
This projet explores the use of machine learning to predict veterinary diseases based on reported symptoms.

## Objectives
 - Clean and preprocess symptom data
 - Train and compare classification models
 - Evaluate performance with clinical relevance in mind

 ## Project Structure

 - `data/` - raw and processed datasets
 - `notebooks/` - exploratory and modeling notebooks
 - `src/` - reuseable preprocessing and modeling code
 - `results/` - figures and metrics

 ## Tools Used

 - Python
 - pandas, scikit-learn, matplotlib, seaborn
